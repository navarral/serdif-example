# -- events ------------------------------------------------------------
Run the following command to create the random events data within Ireland
$ venv/bin/python3.6 EndEventsIreland.py
Enter the number of events and individuals they relate to with your keyboard when asked

Generate a list of urls from the EEA API (https://discomap.eea.europa.eu/map/fme/AirQualityExport.htm)
Download the data files using wget
$ wget -i url-list.txt
